# 风格 A：极简科技风 (Minimal Tech)

> 设计关键词：**克制、留白、单色点缀、工程师气质**

---

## 设计理念

像一份优秀的技术文档——信息密度高，但阅读体验极其舒适。通篇黑白灰为主，仅用一个冷色调点缀色在关键位置「点睛」。追求的是"这个人一看就很靠谱"的第一印象。

---

## 视觉规范

### 配色方案
| 用途 | 色值 | 说明 |
|------|------|------|
| 主文字 | `#1a1a1a` | 接近纯黑，但不刺眼 |
| 次文字 | `#6b7280` | 灰色，用于日期、次要信息 |
| 背景 | `#ffffff` | 纯白 |
| 点缀色 | `#2563EB` | 冷蓝色，仅用于：section 标题、链接、极少量强调 |
| 分割线 | `#e5e7eb` | 极浅灰，section 之间的细线 |

### 字体
```css
font-family: 'Inter', -apple-system, 'PingFang SC', 'Noto Sans SC', sans-serif;
```
- **姓名**：28px, `font-weight: 700`, 颜色 `#1a1a1a`
- **Section 标题**（如"工作经历"）：14px, `font-weight: 600`, `text-transform: uppercase`, `letter-spacing: 2px`, 颜色 `#2563EB`
- **公司/项目名**：15px, `font-weight: 600`
- **正文**：13.5px, `font-weight: 400`, `line-height: 1.7`
- **日期/次要信息**：12.5px, `font-weight: 400`, 颜色 `#6b7280`

### 布局
- **整体**：单栏布局，严格 A4 尺寸（`max-width: 210mm`）
- **页边距**：上下 `20mm`，左右 `25mm`
- **Section 间距**：`24px`
- **条目间距**：`16px`

### 核心视觉元素

1. **Header 区（顶部）**
   - 姓名居左，大号加粗
   - 联系信息紧跟姓名下方一行，用 `·` 分隔（电话 · 邮箱 · GitHub · 城市）
   - 求职意向单独一行，`font-weight: 500`，颜色用点缀色
   - Header 下方一条 `1px solid #e5e7eb` 分割线

2. **Section 标题**
   - 全大写英文 + 中文（如 `WORK EXPERIENCE 工作经历`）
   - 点缀色文字
   - 标题下方无装饰线，靠 `letter-spacing` 和颜色本身做区分
   - 标题上方留 `24px` 间距

3. **公司/项目行**
   - 公司名（加粗） | 职位 | 时间（右对齐，灰色）
   - 用 flexbox 实现左右对齐

4. **正文 bullet points**
   - 使用 `•` 作为列表符号（非默认圆点，手动用 `::before` 伪元素）
   - bullet 颜色用点缀色 `#2563EB`
   - 缩进 `16px`

5. **技能标签**
   - 纯文本排列，用 `·` 分隔
   - **不使用**任何 tag/badge 样式（保持极简）

6. **加粗文字**
   - 简历中的 `**加粗**` 内容用 `font-weight: 600` 呈现
   - 不改变颜色，纯靠字重区分

### 装饰元素
- ❌ 无图标（无 icon）
- ❌ 无色块背景
- ❌ 无阴影
- ❌ 无圆角卡片
- ✅ 唯一装饰：section 间的细分割线

### 打印适配
```css
@media print {
  body { margin: 0; }
  .page { box-shadow: none; }
  a { color: #1a1a1a; text-decoration: none; }
}
```

## 整体气质描述

> 如果这份简历是一个人，他穿的是：**黑色高领毛衣 + 深灰色西裤 + 白色极简运动鞋**。
> 没有一丝多余，但每个细节都恰到好处。一眼就知道是做技术的、靠谱的、审美在线的。
