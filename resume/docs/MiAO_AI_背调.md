- 画像：**典型"资深游戏操盘手 + 社区产品经验"的跨界连续创业者**

## 四、产品演进路径（关键：从游戏 → AI Agent 的 Pivot）

### 第一阶段（2023–2024）：开放世界 voxel 游戏
- 多人体素（voxel）开放世界 + 社交平台
- 类 **Minecraft + Roblox** 的 UGC 游戏 + 社交混合体
- GitHub 开源了 **LARP（Language Agent for Role Play）** 框架 —— 让游戏 NPC 具备 LLM 驱动的智能对话与行为

### 第二阶段（2025–2026 至今）：转向 Personal AI Agent
**miao-ai.com 当前定位**：
- **"个人 AI 助理 / 伴侣型 Agent"**
- 核心理念：**"AI 变得更智能不是重点，AI 变成'你的'才是重点"**
- 强调 Agent **学习用户偏好，随时间进化**
- 走"**稀缺层（Scarce Layer）**"路线 —— 不卷模型能力，卷个性化层

### 战略判断

从"游戏 NPC Agent"到"个人 Agent 助手"，本质是 **把 LARP 积累的角色扮演 / 长期记忆 / 拟人对话的 Agent 能力，从游戏场景迁移到消费级 AI Agent 场景**。

**赛道对标**：Character.AI、Replika、Pi（Inflection，被微软吸收）、Dot（New Computer，被 Sam Altman 收购）等陪伴/个人 Agent 方向。

## 五、组织与文化（极特殊，值得注意）

miao-ai.com 公开表达的运营哲学：

- 🔹 **工程、产品、运营均由 AI 驱动**
- 🔹 **扁平化，无中层管理**
- 🔹 **AI 去发现用户，而非用户去发现 AI**（Let AI discover, not discover for it）

**解读**：这类表述典型于 2025–2026 年 **"AI-native organization"** 潮流（参考 Gumroad、Bland AI 的"全 AI 运营"叙事）。对求职者意味着：

- 🟢 组织敏捷、决策快、个人影响力大
- 🟡 职责边界模糊、对 AI 工具使用能力要求极高
- 🟡 缺少传统 career ladder / 中层晋升路径

---

## 六、风险与关注点

1. ⚠️ **战略二次转型**：从 voxel 游戏到 Personal AI Agent 是一次 **大幅 Pivot**。天使轮资方投的是"游戏+AI"，现在转向"个人 Agent"消费品，需关注 **产品落地情况 + 老资方是否继续跟投**。
2. ⚠️ **赛道红海**：个人 AI Agent 是 2026 年最卷赛道之一（Meta AI、Google Gemini、OpenAI Operator、Anthropic Claude、Manus、Monica、Dot、Replika）。MiAO 的差异化（"个性化稀缺层"）需要产品验证。
3. ⚠️ **商业化路径不清晰**：消费级 Agent 的变现（订阅？广告？IP 衍生？）在全球都尚未跑通。
4. ⚠️ **"AI 运营 + 无中层"文化**：对个人综合能力要求极高，职业发展路径未经市场验证。
5. ⚠️ **产品信息透明度低**：miao-ai.com 官网内容极简（只有 slogan），尚未公开 release 具体产品形态 / 用户量，**可能还处于 Stealth 或 Beta 阶段**。
6. ✅ **正向信号**：
   - 创始人履历扎实（巨人网络高管 + 爆款游戏经验）
   - 资本背书强（红杉中国 + 高榕 + 心动）
   - 团队 LARP 开源项目证明 Agent 技术能力
   - 新加坡架构对美元融资 / 全球市场友好

## 八、最终判断

| 维度 | 评分 | 说明 |
|---|---|---|
| 创始人 | 🟢🟢🟢 | 巨人网络高管，产品感 + 大盘操盘经验强 |
| 资方 | 🟢🟢🟢 | 红杉 + 高榕 + 心动，一线组合 |
| 技术 | 🟢🟢 | LARP 开源框架 + Agent 能力沉淀 |
| 赛道 | 🟡 | 热门但红海，Personal Agent 尚未有赢家 |
| 产品 | 🟡 | 当前产品透明度低，可能 Stealth |
| 商业化 | 🟡 | 消费级 Agent 全球未跑通 |
| 组织 | 🟡 | 全 AI + 无中层，小而精，适合强自驱者 |

**一句话**：**游戏大厂出身的二次创业团队（吴萌 + 巨人网络系）+ 红杉高榕背书 + 从游戏 NPC Agent 转型消费级 Personal Agent**。创始人和资方是亮点，产品尚在早期且赛道竞争激烈；组织文化极特殊（全 AI 驱动 + 无中层），**适合"对 AI-native 工作方式好奇、愿意在消费级 Agent 红海里赌团队基因"的候选人**。
