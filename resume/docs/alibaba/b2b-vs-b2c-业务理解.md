# B2B vs B2C 电商异同 — 结合 Accio 的业务理解

> 面试准备：理解为什么 Accio 选择 B2B 赛道，以及 B2B 场景对 Agent 技术的独特需求。

---

## 一句话结论

**B2B 跨境采购的高复杂度、高信息不对称、长决策链路，使其成为 AI Agent 最有价值的落地场景。Accio 本质上是用 Agent 把"老外贸人的经验"产品化，让全球中小买家也能高效完成专业采购。**

---

## 六个维度对比

### 1. 决策链路

- **B2C**：个人即时决策，看评价、比价格，几分钟到几天完成，冲动型购买占比高。
- **B2B**：组织多角色决策（采购员初筛 → 技术验样 → 质检合规 → 财务审批），周期数周到数月。即使是 Accio 面向的中小买家，仍然是"研究型"购买——需要系统性评估供应商资质、样品、报价。
- **对产品的影响**：B2B 产品必须支持多角色协同流程和长周期信息留存，而非 B2C 的"逛-选-买"线性漏斗。

### 2. 交易流程

- **B2C**：加购 → 下单 → 在线支付 → 物流，标准化程度极高，SKU 价格透明。
- **B2B**：搜索/发现 → RFQ（询盘）→ 多轮报价谈判 → 打样验样 → 合同签订 → 分批交付 → 账期结算，每一步都有来回沟通。没有"一口价"，价格随 MOQ、定制需求、交期、付款方式浮动。
- **Accio 的价值**：把原来需要人工跑展会、发邮件的冗长询盘-报价流程，通过 Agent 自动化压缩到线上几步完成。

### 3. 产品/技术架构

B2B 需要一系列 B2C 完全不涉及的能力模块：

| B2B 独有需求 | 说明 |
|-------------|------|
| 供应商资质评估 | 工厂认证、验厂报告、贸易信用评级 |
| 合规与关税引擎 | 不同目的国进口法规、关税计算、禁运清单 |
| 多币种与汇率结算 | 跨国交易基础设施 |
| MOQ/阶梯定价 | 价格不是固定的，随量变化 |
| 样品与定制化管理 | B2B 商品不是固定 SKU，是"可配置的供给能力描述" |

### 4. AI/Agent 应用差异（核心重点）

- **B2C 的 AI**：主要做**推荐引擎**（猜你喜欢）和搜索排序，本质是信息过滤，用户自己做决策。AI 是锦上添花。
- **B2B 的 AI**：需要扮演**采购助手/代理人**，因为用户面对的不是"选哪个"的简单问题，而是"怎么找靠谱供应商、怎么评估、怎么谈好价格"的复杂多步任务。AI Agent 是不可替代的。

**B2B 为什么更需要 Agent 而非 Chatbot：**
- **意图理解**：从模糊需求提取采购规格（"我想找能做硅胶厨具的工厂，FDA 认证，MOQ 不超过 500"）
- **多步规划**：拆解采购流程为可执行步骤
- **工具调用**：查供应商数据库、计算关税、生成 RFQ
- **多轮记忆**：跟踪谈判状态和历史上下文

**Accio 选 B2B 的核心逻辑**：B2C 推荐系统已经很成熟，Agent 增量价值有限；B2B 跨境采购中，Agent 能把原本需要资深外贸业务员才能完成的工作自动化——这是真正的不可替代价值。

### 5. 数据和搜索差异

- **B2C 搜索**：具体商品（"iPhone 16 手机壳"），query 短、标准化，搜索结果直接展示 SKU 列表。
- **B2B 搜索**：模糊的采购需求，query 长、包含多维约束条件，需要匹配的是供应商的生产能力而非单一商品。
- **Accio 的搜索架构**：不能是传统关键词匹配，必须是**语义理解 + 结构化约束过滤 + 供应商能力图谱**的混合检索——天然的 RAG + Agent 应用场景。

### 6. 用户画像

- **B2C 用户**：个人消费者，情绪驱动（促销、种草、冲动），容错成本低（不满意退货），不需要专业知识。
- **B2B 用户**：企业采购员或中小企业老板，**ROI 和风险驱动**，一次采购失误可能意味着数万美元损失和供应链断裂，极度重视供应商信誉和产品合规性。
- **Accio 的精准切入点**：全球中小企业主缺乏专业采购团队和供应链资源，Agent 本质上是在充当他们**请不起的专业采购经理**。

---

## 总结表

| 维度 | B2C | B2B（Accio） |
|------|-----|-------------|
| 核心价值 | 缩短"逛"到"买"的路径 | 降低跨境采购的专业门槛 |
| AI 定位 | 推荐/搜索优化（锦上添花） | 采购代理人（不可替代） |
| Agent 必要性 | 低（简单意图即可满足） | 高（多步骤、多约束、多轮交互） |
| 竞争壁垒 | 流量和供应链效率 | 供应商数据图谱 + Agent 能力 |
| 用户决策 | 冲动型，分钟级 | 研究型，周/月级 |
| 交易标准化 | 高（一口价、标品） | 低（谈判、定制、阶梯价） |
