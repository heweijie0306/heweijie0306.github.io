P / 01

Pageon PPT — Data‑structure rewrite

Initiator & Solution Owner 2025.11 — 2026.01

Legacy XML + React kept ceiling‑capping visual quality. I proposed and validated a fundamentally different shape: ship native HTML straight from the model.

Rewrote the data layer. I independently surfaced the bottleneck and proposed HTML‑direct + front‑end post‑processing — using context engineering to make the LLM emit native HTML/CSS instead of XML, with DOM Parser + tiptap handling rich‑text editing and drag interactions.

Drove production rollout. Built the architecture demo alone, pitched it, and the team adopted it as the long‑term technical roadmap. After shipping, conversion held at 5%+, the visual ceiling and Agent capability pulled ahead of Gamma, and the surface stretched to Poster, Infographic, and Social Media.

P / 02

Pageon PPT — Page Agent Infra

Core Engineer 2025.07 — Present

The full evolution of our PPT Agent infrastructure — from a hand‑rolled ReAct loop to the Claude Code Agent SDK, with MCP bringing external data in.

Designed the Agent toolchain. Built Tool Calling + ReAct Loop in‑house with paged read, file edit, HITL interruption, and auto‑recovery. Against unstable LLM diffs I shipped 7 Search & Replace strategies, including XML structural matching — dropping edit‑failure rate sharply.

Led the migration to Claude Code SDK. Rebuilt the adapter layer on top of its native File System and Bash Tool; in parallel, developed an MCP stdio server and HTTP‑MCP entry so users can pull from GitHub, Notion, and their own cloud data sources.

Pageon MG Video — Animation Agent

Core Engineer 2026.01 — Present

Built the Agent runtime. Stood up the runtime on PiAgent SDK + just‑bash on Vercel — no container, no WASM. Wrote custom tools — validate (Babel compile + static check), create_image, TTS — so the Agent can emit TSX code and the user previews it live.

Designed the business‑driven layer. Designed a Skills system that turns 10 animation domains (3D, transitions, motion design, …) into runtime‑injected domain knowledge driving the Agent. Multi‑scene composition + AWS Lambda cloud rendering for export.
